Build a premium, modern, interactive full-stack website for a lighting brand named “AREV Lights”.

PROJECT GOAL:
Create a high-end business website for a lighting / electrical / decorative lights company in the same market segment as premium lighting brands. The website should feel modern, elegant, trustworthy, and visually rich. It must not look like a generic template. It should be conversion-focused and all major website content must be manageable through an admin dashboard.

IMPORTANT:
This is not a simple static website.
This is a production-ready full-stack CMS-driven business platform.

TECH STACK:
- Frontend: Next.js (App Router) + TypeScript
- Styling: Tailwind CSS
- UI quality: premium, modern, interactive, business-class
- Animations: Framer Motion or equivalent smooth motion system
- Backend: Node.js / Next.js server actions or API routes
- Database: MongoDB with Mongoose
- Media Storage: Cloudinary or equivalent
- Auth: secure admin authentication using JWT / NextAuth / session-based auth
- Forms: robust validation
- Image optimization: must be handled properly
- SEO: production-ready metadata and schema basics

DESIGN DIRECTION:
The site should feel like a premium lighting and architectural product brand.
Use clean spacing, luxury visual hierarchy, elegant typography, soft shadows, glassmorphism only if subtle, and motion that feels expensive.
The UI should be interactive but not childish.
Make the experience feel polished like a top-tier brand website.
